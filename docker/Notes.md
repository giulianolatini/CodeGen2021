# Blazor App

Blazor Intro
[Blazor Tutorial | Build your first app](https://dotnet.microsoft.com/learn/aspnet/blazor-tutorial/intro)

## Create

```powershell
dotnet new blazorserver -o BlazorApp --no-https
git a BlazorApp
mkdir docker
code docker\Dockerfile
git a docker
```

## Run

```powershell
cd BlazorApp
dotnet watch run
```
